1 · Executive Summary

The Applied AI Training Program equips 18 participants with production-grade skills in AI agent engineering, prompt design, tool calling, and deployment to modern cloud platforms.

Our Hybrid Split delivery model — two groups of 9, blending face-to-face and remote sessions — reduces total cost by 40% versus a pure on-campus program while preserving the hands-on rigor your faculty identified as critical. Over 24 training days across 6 weeks, each participant ships a working AI agent to production, with a public GitHub portfolio and measurable skill outcomes.

This proposal outlines scope, timeline, investment, and acceptance terms. Total engagement value is AED 615,000, inclusive of curriculum, delivery, and joint certification.

2 · Program Scope

The program is structured in four progressive modules, each building on the last. Participants move from fundamentals to production deployment over the 6-week span.

# Module Focus Outcome
1 Foundations Prompt engineering, LLM fundamentals, tool calling Working prompt library with evals
2 Agent Design Multi-step agents, state management, memory patterns Prototype agent with 3+ tools
3 Production Deployment to Vercel, Cloudflare Workers; observability Live agent deployed to production URL
4 Integration API design, security, cost controls, team handoff Documented, maintainable codebase
